# RTM Execution Hub

Single HTML + Firebase + GitHub Pages
สำหรับทีม RTM FMCG 4 คน (Manager + A/B/C)

## รัน local

Double-click `index.html` → เปิดใน Chrome
ไม่มี build step · ไม่มี Node · ไม่มี npm

## Deploy

Push ไปที่ branch `main` → GitHub Pages serve อัตโนมัติ
(เปิดที่ Repo Settings → Pages → Source: main / root)
